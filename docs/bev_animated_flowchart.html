BevProcessor — main loop (500×500×4 = 1,000,000 iterations)
A
World coordinates
worldX = 5 − row×0.02 m  ·  worldY = −5 + col×0.02 m
Origin = rear axle · +X forward · +Y right
↓
D
Blend weight (early exit)
cosA = (cosYaw·wx + sinYaw·wy) / dist  ·  if cosA≤0 skip
w = cosA^1.5 / dist — checked FIRST for speed
↓
B
Camera-space transform
[xc, yc, zc] = R × [worldX, worldY, CAM_HEIGHT]
3×3 matrix multiply — 9 multiplications
↓
C
Mei fisheye projection
r3d = √(xc²+yc²+zc²)  ·  mzXi = zc/r3d + ξ
u = fx·(xc/r3d)/mzXi + cx  ·  v = fy·(yc/r3d)/mzXi + cy
↓
E
Bilinear interpolation
4-neighbour blend · du=u−floor(u) · dv=v−floor(v)
c = c00·(1-du)(1-dv) + c01·du(1-dv) + c10·(1-du)dv + c11·du·dv
↓
F
Weighted accumulation
bevR[i] += w·R  ·  bevG[i] += w·G  ·  bevB[i] += w·B  ·  totalW[i] += w
Runs for all 4 cameras · same pixel index i
↓
